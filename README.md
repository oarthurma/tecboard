## 📋 Sobre o Projeto

O **TecBoard** é uma landing page desenvolvida como projeto prático do curso de HTML e CSS da Alura. O objetivo foi colocar em prática os conceitos fundamentais de estruturação semântica com HTML5, estilização com CSS3 e publicação com Git, GitHub e GitHub Pages.

***

## 🎓 O que foi aprendido no curso

| Módulo | Conteúdo |
|--------|----------|
| 🛠️ Ambiente | Instalação e configuração do VS Code, Chrome DevTools |
| 🏗️ HTML | Tags semânticas, hierarquia de títulos, listas, links, imagens |
| 🎨 CSS | Box Model, fontes, cores, responsividade mobile-first |
| 🔀 Git & GitHub | Versionamento, commits, branches, fluxo profissional com PRs |
| 🚀 Deploy | GitHub Pages, hospedagens pagas, domínios, DNS |

***

## 🚀 Tecnologias

- **HTML5** — estrutura semântica da página
- **CSS3** — estilização, responsividade e layout
- **Git** — controle de versão
- **GitHub Pages** — hospedagem e deploy contínuo
- **VS Code** — editor de código
- **Figma** — referência de design
